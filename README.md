# 🚀 Elevvo Internship Portfolio

Welcome to my frontend development portfolio! This repository contains the projects I built during my 1-month internship track with Elevvo. 

My core focus throughout these tasks was writing clean, well-documented, and structurally sound code, while stripping away unnecessary boilerplate to keep the architecture secure and easy to read.

## 📁 The Projects

Here is a breakdown of the 4 core tasks completed in this repository:

### 1. Task 8: Industry-Level Job Tracker
An advanced React application designed to track job search progress.
* **Tech Stack:** React, Context API, React Router, LocalStorage.
* **Highlights:** Features a custom UI, dynamic status badges, and persistent browser storage so user data is never lost on refresh. All state is managed globally using the Context API to avoid prop-drilling.

### 2. Task 4: Personal Blog Dashboard
A dynamic blog interface built to master React state and conditional rendering.
* **Tech Stack:** React, CSS.
* **Highlights:** Includes real-time title searching and category filtering working together seamlessly to display specific data arrays.

### 3. Task 3: Interactive Pricing Table
A highly reusable, component-driven pricing page.
* **Tech Stack:** React, CSS.
* **Highlights:** Uses a single "Master Card" component mapped across an array of data to dynamically generate multiple pricing tiers efficiently, highlighting the featured plan automatically.

### 4. Task 2: Responsive Contact Form
A clean, simulated contact form with strict front-end validation.
* **Tech Stack:** HTML, JavaScript, Bootstrap.
* **Highlights:** Uses JavaScript and Regular Expressions (Regex) for email and empty-field validation to ensure data integrity before allowing a simulated submission.

## 🛠️ How to Run Locally
To test any of the React projects (Tasks 3, 4, or 8) on your own machine:
1. Clone this repository.
2. Open your terminal and `cd` into the specific project folder (e.g., `cd task-8-job-blog`).
3. Run `npm install` to install the required dependencies.
4. Run `npm run dev` to start the local Vite development server.
